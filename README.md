# ipchecker
A simple utility for checking info about your server IP-address

# Features
- Show IPv4 and IPv6 (if supported) addresses
- Display country, city, timezone and ASN
- Color-coded output

# Usage
For local usage, download the script:

```
wget https://raw.githubusercontent.com/wattamuffin/ipchecker/refs/heads/main/ipchecker.sh 
chmod +x ipchecker.sh
```
Or you could run the script directly from GitHub:
```
bash -c "$(curl -fsSL https://raw.githubusercontent.com/wattamuffin/ipchecker/refs/heads/main/ipchecker.sh)"
```
